xyts (xy time series)

Programa para grabar muchos gráficos xy de series temporales con los datos almacenados en una BDD ms access
Permite dibujar la evolución temporal de una variable de un punto y de la misma variable en otros puntos relacionados con el primero; también se puede dibujar la misma u otra variable en un gráfico situado debajo de la evolución anterior. 

HISTORIA
Es la evolución del programa dislin27. Este programa utilizaba: el módulo pywin32 para acceder a la bdd ms access; la librería dislin para realizar los gráficos
La versión actual ya no utiliza estas librerías: pyodbc en vez de pywin32; matplotlib en vez de dislin

INSTALACIÓN

1. Descomprimir el fichero zip en el directorio seleccionado

EJECUCIÓM

Requisitos previos
python 3.7
Dependencias: numpy, matplotlib y pyodbc

Se ha desarrollado con Anaconda, para ejecutarlo

Los gráficos están agrupados en proyectos en el fichero xyts.xml; si quieres hacer uno nuevo fíjate en los que ya hay creados

1 Abrir Anaconda, spyder y ejecutar xyts_main.py

2. Abrir una ventana de conda; cd al directorio de instalación; python xyts_main.py
